# Implementation Plan

- [x] 1. テキスト後処理パイプラインへの句点付与ステップ追加
- [x] 1.1 既存の後処理パイプラインの末尾に句点付与ロジックを実装する
  - 後処理パイプラインの最終ステップとして、テキスト末尾への句点付与処理を追加する
  - 末尾文字が「。」「！」「!」「？」「?」のいずれかに該当する場合は句点を付与しない
  - 空文字または空白のみのテキストには句点を付与しない
  - 読点（、）への操作は一切行わない
  - 既存のステップ1〜4（空白除去・フィラー除去・スペース正規化・繰り返し集約）が完了した後に実行されることを確認する
  - _Requirements: 1.1, 1.2, 1.3, 1.4, 2.1, 2.2, 2.3, 3.1, 3.2_

- [x] 2. 句点付与ロジックの単体テスト追加
- [x] 2.1 句点付与の境界値をカバーする単体テストを追加する
  - 句点なし入力（例：「おはよう」）に対して末尾に「。」が付与されることを検証する
  - 既に「。」で終わる入力に対して重複付与されないことを検証する
  - 「！」「!」「？」「?」で終わる入力に対して句点が付与されないことを検証する
  - 空文字入力に対して空文字がそのまま返ることを検証する
  - 空白のみ入力に対して空白のみが返ることを検証する
  - 読点（、）を含む入力に対して読点が変化せず末尾に「。」のみ付与されることを検証する
  - _Requirements: 1.1, 1.2, 1.3, 2.1, 2.2, 3.3_
